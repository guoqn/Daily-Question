# localhost 与 127.0.0.1 有什么区别



::: tip Issue 
 欢迎在 Issue 中交流与讨论: [Issue 365](https://github.com/shfshanyue/Daily-Question/issues/365) 
:::

::: tip Author 
回答者: [hedongxiaoshimei](https://github.com/hedongxiaoshimei) 
:::

localhost 可以更改后指向其它地址。
127.0.0.1 是约定的本机IP。
类似互联网中域名与服务器IP的关系。
